# OpenEnv Hackathon Submission

## Overview
This repository contains the submission for the Meta OpenEnv Hackathon. It implements a Reinforcement Learning environment using the OpenEnv framework.

## Prerequisites
Python 3.10+
Docker Desktop

## Setup Instructions

1. Clone the repository:
   git clone https://github.com/A-stha-M/openenv-submission.git
   cd openenv-submission

2. Activate the virtual environment:
   venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt
